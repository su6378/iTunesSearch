# 🎵 iTunesSearch

iTunesSearch 프로젝트는 iTunes Search API를 통해 해당 키워드로 검색한 트랙 정보들을 화면에 페이징을 통해 보여주고, 즐겨찾기 화면에서 트랙 정보 화면에서 즐겨찾기한 트랙을 확인할 수 있습니다.
